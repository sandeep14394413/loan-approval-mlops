# How to get the dataset

1. Open the Kaggle dataset page:
   https://www.kaggle.com/datasets/yaminh/applicant-details-for-loan-approve?resource=download

2. Download the CSV file (you need a free Kaggle account).

3. Rename the file if needed and place it at:

```bash
data/raw/applicant-details-for-loan-approve.csv
```

4. The file is listed in `.gitignore` — it will NOT be committed to git.
